1. Качаешь сам NGROK https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-windows-amd64.zip
2. CMD -> cd [путь_до_файла_ngrok.exe] -> ngrok.exe http [port] 8080
3. Где написано Forwarding URL [Пример на конце .ngrok-free.app] - это и есть сама ссылка на WEB APP
4. Заходишь в @Botfather - пишешь /myapps - выбираешь своё приложение и нажимаешь Edit Web App URL и вставляешь ссылку с пункта 3.
